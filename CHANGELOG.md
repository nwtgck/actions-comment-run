# Changelog
All notable changes to this project will be documented in this file.

## [Unreleased]

## [2.0.0] - 2023-05-05
### Changed
* Updates the default runtime to node16
* Use @actions/core 1.10.0
* Use @actions/exec 1.1.1
* Use @actions/github 5.1.1
* Use node-fetch 3.3.1
* Update dependencies

## [1.1.3] - 2020-04-07
### Changed
* Update dependencies

## [1.1.2] - 2020-04-03
### Changed
* Update dependencies

## [1.1.1] - 2020-03-23
### Fixed

## [1.1.0] - 2020-03-23
### Changed
* Update dependencies

### Added
* Support top-level await

## [1.0.3] - 2020-03-10
### Changed
* Add eyes reaction when started and after finished successfully add +1 reaction and remove the eyes 

## [1.0.2] - 2020-03-10
### Changed
